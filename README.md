实现的功能：

系统分为管理员、媒体用户（上传视频）、观众用户，三种角色。

管理员：观众用户管理、媒体用户管理、公告管理、分类管理、视频管理等；

观众用户：查看视频、查看公告、登录注册、收藏等；

媒体用户：发布视频、登录注册等。

使用的技术：

后端框架：SSM（Spring、SpringMVC、Mybatis），MySQL数据库、maven依赖管理等；

前端技术：Bootstrap、html、css、JavaScript、JQuery、VUE等。
